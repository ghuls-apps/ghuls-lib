# ghuls-lib
[![Gem Version](https://badge.fury.io/rb/ghuls-lib.svg)](https://badge.fury.io/rb/ghuls-lib)

The library Ruby Gem for GHULS

## Installation
### Ruby Gems
```
gem install ghuls-lib
```

### Bundler
Add this line to the application's Gemfile:
``` ruby
gem 'ghuls-lib'
```

And then execute:
```
$ bundle
```

## Usage
``` ruby
require 'ghuls-lib'
```
